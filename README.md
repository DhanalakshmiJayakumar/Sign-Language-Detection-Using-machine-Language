# Sign Language Detection Using Machine Learning

# Project Overview

This project uses Machine Learning to recognize sign language hand gestures from images or live webcam input. The system predicts the corresponding alphabet or word and displays it as text, helping improve communication for people with hearing or speech impairments.

---

# Features

- Real-time hand gesture detection
- Sign language recognition
- Live webcam prediction
- Converts hand signs into text
- Easy-to-use interface

---

# Technologies Used

- Python
- OpenCV
- NumPy
- Scikit-learn / TensorFlow (depending on your model)
- Jupyter Notebook

---

# Workflow

1. Collect sign language dataset
2. Preprocess images
3. Train Machine Learning model
4. Test the model
5. Predict hand gestures
6. Display detected sign
